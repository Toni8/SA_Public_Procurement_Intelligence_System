# Public Procurement Intelligence System - South Africa 🇿🇦

A data-driven analytics platform that transforms South African eTenders data into actionable intelligence for procurement officials, suppliers, and policy analysts.

### Project Overview
This project analyzes historical public tender data from the South African eTenders portal (OCDS format) to uncover spending patterns, forecast contract values, detect anomalies, and identify high-opportunity sectors and provinces.

### Key Features
- **Data Processing**: Inspection, cleaning, and Exploratory Data Analysis (EDA)
- **Analysis**: Advanced SQL queries on 8 interrelated datasets
- **Machine Learning**: Contract value forecasting models
- **Insights**: Anomaly detection and opportunity identification
- **Visualization**: Interactive Power BI dashboards

### Datasets
- **Main Tenders**: 43,147 records  
- **Awards**: 8,181 records  
- **Suppliers, Contracts, Documents, Parties & Tenderers** (total ~68,000 records)

### Target Users
- Procurement Officials  
- Business Development / Supplier Teams  
- Policy Analysts & Government Entities  
- SMEs and Contractors

### Technologies Used
Python | Pandas | SQL | Scikit-learn | Power BI | OCDS Data Standard

---

**Goal**: Enable data-informed decision making in South African public procurement.

*Prepared using 2025 eTenders South Africa data*
